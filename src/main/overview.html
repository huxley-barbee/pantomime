Pantomime is a Java library for manipulating and storing MIME messages.

As an alternative to JavaMail it is

  1. Easy to learn and convenient to use.
  2. Memory-efficient. Able to load messages of any size without exhausting memory.
  3. Robust. Better adapted to handle mish-mash of real world messages that often do not comply with RFC.

To get started please read the manual available at http://barbee.bitbucket.org/pantomime/manual/pantomime_manual.html.
